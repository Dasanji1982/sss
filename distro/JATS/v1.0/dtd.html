Journal Publishing Tag Set

Version 1.0 DTD

The DTD is the format from which all other schemas are derived. It is the only schema that is intended for maintenance.

Getting the Files

The DTD files are available on the anonymous FTP: ftp://ftp.ncbi.nih.gov/pub/jats/publishing/1.0/.

The schema is also available at the following stable URL: http://jats.nlm.nih.gov/publishing/1.0/JATS-journalpublishing1.dtd

Schema Structure

The main module of the DTD is JATS-journalpublishing1.dtd. It imports the subordinate modules. For the schema to work, all modules must be in the same directory.
